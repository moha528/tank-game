# Tank Arena Prototype

## Démarrer localement

1. Ouvre un terminal dans ce dossier.
2. Lance un serveur statique :

   ```bash
   python -m http.server 8000
   ```

3. Ouvre ton navigateur sur `http://localhost:8000/index.html`.

## Commandes en jeu

- Déplacement : ZQSD / WASD / Flèches
- Tirer : Espace
- Changer d'arme : 1 / 2
- Changer de tank : boutons dans la colonne de gauche
